# AGENTS.md

Instructions for coding agents working on the `clast` repo itself.

## Conventions

- **Dev shell**: If an expected tool is unavailable on PATH, run `make deps-check` to see the expected tools and whether they are available. The tools should all be available via Nix via `direnv exec . <tool>`.
- **Run `make test` and `make lint` before committing.**
- **Conventional commits**: `feat:`, `fix:`, `docs:`, `chore:`, `test:`, `refactor:`, `ci:`.
